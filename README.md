Website cá nhân
Website cá nhân phục vụ đăng bài blog kỹ thuật, giáo trình, nghiên cứu và tài liệu chuyên ngành điện tử – sử dụng Next.js 15, Sanity.io (CMS) và Tailwind CSS.
- Features
✔️ Hệ thống blog kỹ thuật với bài viết, tác giả, chuyên mục
✔️ Sanity Studio CMS — giao diện quản trị mạnh mẽ, realtime
✔️ SEO tốt — hỗ trợ Next.js App Router
✔️ Tương thích Mobile & Desktop (Responsive)
✔️ UI TailwindCSS, dễ tuỳ biến
✔️ Dữ liệu lấy từ Sanity bằng GROQ
✔️ Deploy dễ dàng lên Vercel & Sanity Hosted
- Tech Stack
Frontend

App Router
TailwindCSS 4+
Typescript (khuyến nghị)
CMS
Sanity.io
